HTML Tags

<h1> - <h6>
Headings. The number denotes heading size and hierarchy (h1 is the largest, h6 is the smallest).
<p>
Paragraph. Block-level tag used to contain text.
<img>
Image. Uses the src attribute to specify the image path and the alt attribute for alternate text.
<a>
Hyperlink. Uses the href attribute to specify the URL.
<ol>, <ul>, and <li>
Ordered list, unordered list, and list item. Ordered and unordered lists must only have list items as child tags. List items can have any child tags.
